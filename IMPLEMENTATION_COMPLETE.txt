### How to Run

Terminal 1 - Backend:
```bash
pip install -r requirements_api.txt
python run_api.py
```
Backend will start on http://localhost:8000

Terminal 2 - Frontend:
```bash
cd web
npm install
npm run dev
```
Frontend will start on http://localhost:3000

### Architecture

Backend:
- FastAPI with background tasks
- In-memory job storage (MVP)
- Reuses ReportOrchestrator
- Reuses config_models validation
- Files saved to output/api/

Frontend:
- React 18 with TypeScript
- Vite for build/dev
- Zustand for state management
- Axios for API calls
- Tailwind CSS for styling
- localStorage for draft persistence

### Known Limitations (MVP)

1. Job storage is in-memory (resets on server restart)
2. No authentication/authorization
3. No file cleanup (files accumulate in output/api/)
4. No database persistence
5. Single-user (no multi-tenancy)

These are acceptable for MVP and can be enhanced in future iterations.
